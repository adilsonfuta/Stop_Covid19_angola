ENTENDA COMO NOSSA API FUNCIONA

O Webservice do coronavirus foi criado gratuitamente para que a populacão possa a ter acesso dos dados atraves de aplicações desenvolvidas por programadores ou por pessoas de conhecimentos necessarios. Pelo fato da facto de ser gratuito pedimos a colaboração de todos informando " casos suspeitos", contamos com a sua ajuda, clique aqui.

A estrutura da nossa api possui dois metodos sendo eles o "listar" e o "encontrar" , como o proprio nome sugere o listar retorna uma lista de todos as provincias em ordem crescente envolvendo os numeros de casos registrados, ja o encontrar necessita de um parâmetro, pois trata de um mêtodo de pesquisa, ou seja, o retorno depende do parametro passado via URL. Para ambos os casos o resultado sera uma lista de dados retornado via json.
